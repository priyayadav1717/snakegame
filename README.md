# 🐍 Snake Game

Snake Game is a classic arcade game developed using Python and the Turtle graphics library. Players control a growing snake, collect food to increase their score, and avoid collisions with the walls and the snake's own body. The project demonstrates core programming concepts such as object-oriented programming, event handling, collision detection, and game loop implementation.

## Features

* 🐍 Classic Snake gameplay
* 🍎 Random food generation
* 📈 Real-time score tracking
* ⌨️ Keyboard controls for snake movement
* 💥 Collision detection with walls and snake body
* 🔄 Automatic game-over handling and restart support (if implemented)

## Tech Stack

* **Language:** Python
* **Library:** Turtle Graphics
